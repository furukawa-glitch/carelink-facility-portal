# 日次インボックス（`data/inbox_daily/`）

スタッフがアップロードした**当日分の録音・メモ**を一時置きし、夜間バッチで SOAP 化・カイポケ用CSVへ集約する前提のフォルダ。

## 推奨ファイル命名

```
{patient_id}__{YYYYMMDD}__{staff_initials}__{seq}.{wav|mp3|m4a|txt}
```

例: `P000001__20260411__YT__01.wav`

## メタデータ（任意）

同じベース名の `.meta.json` を置く:

```json
{
  "patient_id": "P000001",
  "visit_date": "2026-04-11",
  "staff": "YT",
  "note": "夜間訪問"
}
```

## セキュリティ

- 本フォルダ内の**音声・文字起こしは Git に含めない**（`.gitignore`）。
- 処理後は**暗号化ストレージ**へアーカイブし、インボックスからは削除する運用を推奨。
