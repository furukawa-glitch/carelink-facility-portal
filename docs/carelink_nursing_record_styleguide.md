# CareLink 訪問看護 — 記録の書き方マスター（文体・SOAP・用語）

本書は、CareLink（訪問看護ケアサポート）で用いている**記録書Ⅱ・計画書・フェースシート**の実務パターンから、**体裁・SOAP構成・用語の型**のみを抽象化したものである。**個人を特定する情報や、特定利用者の全文記録は含めない**（原本PDFはオーナー管理・Git非掲載）。

AI は録音から記録を起こす際、**本書＋`docs/nursing_master_guideline.md`＋`Instructions.md` §5** をあわせて参照する。

---

## 1. 帳票と記録ブロックの対応

| 帳票 | 役割 | AIが真似るべき「粒」 |
|------|------|---------------------|
| フェースシート（記録書Ⅰ） | マスタ情報・装着器具・ADL・依頼目的 | **固定コンテキスト**（日々のOと矛盾させない） |
| 訪問看護計画書（看護） | 長期/短期目標、観察・ケア項目、材料 | **当日のO/Aで触れるべき観点のチェックリスト** |
| 褥瘡対策看護計画書（医） | DESIGN-R、危険因子 | 褥瘡・皮膚関連訪問時の**記述粒度** |
| 記録書Ⅱ | バイタル、観察事項、実施、**観察記録（SOAP）** | **本文の骨格・語彙・順序** |
| 訪問看護報告書 | カレンダー記号、病状経過要約 | 月次の**要約のトーン**（日次SOAPとは別） |

## 2. 記録書Ⅱ — 観察記録（SOAP）の型

### 2.1 マーカーと段落

- 観察記録は **`【S】`** **`【O】`** **`【A】`** **`【P】`** で開始する（全角括弧・全角カッコの運用は事業所様式に合わせる）。
- **【S】** は短くてよい（「・・・」のみの日もある）。家族発言を入れる場合は**引用風**にし、解釈は【O】【A】に分離する。
- **【O】** が最も長い。**「訪問時〜」** で時間的文脈を置く。

### 2.2 【O】の推奨記述順（重症在宅例の型）

実務サンプルに現れた**読みやすい順**を推奨する（すべて該当するとは限らない）。

1. **訪問時の全身像**：開眼・追視・意思疎通、姿勢、掛け物・室温調整の実施など  
2. **顔色・皮膚・末梢**：紅潮、体熱感、冷感、チアノーゼの有無  
3. **呼吸**：様式、痰絡み、喘鳴、肺雑音  
4. **吸引**：経路別（**口腔・鼻腔**／**気切部** 等）、痰の**色・粘稠度・量**（多量／中等量／少量 等）  
5. **気管カニューレまわり**：**カフ圧**（hPa 等）、**Yガーゼ**の汚染と交換、**気管内筒**の汚染と交換、**人工鼻**・加湿  
6. **口腔**：唾液貯留、口腔ケア、**プロペト**等の塗布、口唇乾燥  
7. **腹部・消化管**：腹壁緊張、腹満、腸蠕動音（記録上部の観察事項欄と整合）  
8. **その他当日実施**：体位変換、オムツ、経管栄養手技の有無（実施した事実のみ）

※ **数値（カフ圧、SpO2、体温など）は音声・実測がある場合のみ**記載する（捏造禁止）。

### 2.3 【A】

- 【O】で述べた事実の**要約・看護課題の言語化**。  
- 推定は **「〜の可能性」** 等にとどめ、断定的な医学診断は避ける（記録の役割の範囲で）。

### 2.4 【P】

- 次回までの観察重点、家族指導の要点、医師・他職種連絡が必要な条件。  
- **計画書のケア項目**と整合するよう、用語を揃える（例: 経管栄養、NG固定、気管カニューレ管理）。

## 3. 用語・表記の辞書（CareLinkで頻出）

### 3.1 観察・状態

| 用語・表現の例 | メモ |
|----------------|------|
| 訪問時、開眼、追視なし／あり | 意識・反応の客観 |
| 顔面紅潮、体熱感、下肢冷感 | 熱・循環 |
| チアノーゼなし | 否定も明示 |
| 呼吸穏やか、痰絡み、喘鳴、肺雑音 | 呼吸系 |
| 腸蠕動音：良好／緩慢 | 計画書・観察事項欄と揃える |

### 3.2 気道・吸引・気切

| 用語・表現の例 | メモ |
|----------------|------|
| 口腔・鼻腔より吸引、気切部より吸引 | **経路を分けて書く** |
| 白色痰、透明痰、粘稠痰、多量／中等量／少量 | 性状＋量 |
| カフ圧〇〇hPaに調整 | 単位付き |
| Yガーゼ汚染にて交換 | 交換理由を一言 |
| 気管内筒汚染〇〇にて交換 | 同上 |
| 人工鼻、加湿 | 在宅呼吸管理 |
| 下顎整復、開口、唾液貯留 | 該当利用者の文脈では頻出しうるが、**他利用者では当てはまらない** |

### 3.3 栄養・経管・NG

| 用語・表現の例 | メモ |
|----------------|------|
| 経管栄養実施・管理 | 計画書の表現と揃える |
| NGチューブ、固定、メモリ | `nursing_master_guideline` の `GASTRO_NG` と連動 |
| 注入、逆流 | |

### 3.4 皮膚・褥瘡

| 用語・表現の例 | メモ |
|----------------|------|
| 褥瘡予防、除圧、スキンケア | |
| DESIGN-R、滲出、部位（仙骨・踵等） | 褥瘡計画書と揃える |

### 3.5 バイタル表記

- 体温 **〇〇℃**、血圧 **〇〇/〇〇 mmHg**、脈拍 **〇〇回/分**、SpO2 **〇〇％**、不整脈 **なし/あり**、副雑音 **なし/あり**  
- 記録上部の**表形式欄**と、SOAPの**本文**で数値が食い違わないよう注意（音声に両方ある場合は整合確認）。

## 4. 「観察事項」欄（表形式）との役割分担

- 記録書Ⅱ上部の**観察事項**は、しばしば **ラベル：値** の短い項目（意識状態、浮腫、腸蠕動音、疼痛スコア、排便 等）。  
- **SOAPの【O】** は、訪問の**ストーリー**として補足する。  
- AI は両方を出力する場合、**同じ事実の矛盾**（例: 腸蠕動音が欄と本文で逆）を検出したら**警告**する。

---

## 5. 禁止・慎重領域（品質と法務）

- 録音に**ない**数値・実施・家族発言を**埋めない**。  
- 算定加算の有無を**断定**しない（計画書・報告書の備考は**参照**にとどめる）。  
- 個人情報（家族氏名・連絡先・住所）は、**フェースシートに任せ**、SOAPに不要なら書かない。
