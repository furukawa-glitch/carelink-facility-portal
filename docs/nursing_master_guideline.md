# 訪問看護 — 必須記録マスター（ガイドライン辞書）

CareLink の**看護DX**（音声→記録、記録品質チェック）で参照する、処置・管理別の**必須記録観点**と**チェック用キーワード**のマスターである。

**記録の文体・SOAPの型**は **`docs/carelink_nursing_record_styleguide.md`** と併用する。

**診療報酬・監査の観点（令和8年度改定のたたき台）**は **`docs/visit_nursing_ai_audit_manual_r8.md`** を参照する（告示本文の確認が前提）。

---

## 0. 文書の位置づけ（必読）

- 本マスターは、訪問看護実務の教科書・各種ガイドラインおよび**診療報酬上の記載・算定に関する一般的要件**を踏まえた**社内用のたたき台**である。
- **診療報酬は年度ごとに改定される**。算定可否・記載要領の最終判断は、**最新の告示・通知・都道府県の取扱い**および**看護師長・診療所の方針**に従うこと。
- AI・システムは**臨床判断の代行はしない**。不足警告は「再確認の手がかり」であり、**記載の真偽・適切性の保証ではない**。

---

## 1. マスターの構成（辞書スキーマ）

各「カテゴリ」ブロックは、次の要素を持つ。

| 要素 | 説明 |
|------|------|
| **カテゴリID** | システム連携用の固定識別子 |
| **対象処置・管理** | 何の記録か |
| **必須記録項目** | 評価・観察・実施内容として残すべき観点（箇条書き） |
| **キーワード群** | 音声・下書きテキストに**含まれているかを機械的に見るための語**（同義語・略語を含む） |
| **充足ルール** | `ALL`＝全グループから最低1語、`ANY_OF`＝いずれかの語があれば可、など |
| **SOAP 補完の指針** | 専門用語・評価枠の**書き方の型**（事実の捏造は禁止） |
| **診療報酬・記録上の注意** | 算定・監査で問われやすい観点（一般論） |

---

## 2. カテゴリ別マスター

### 2.1 `IV_FLUID` — 点滴・静脈ライン・中心静脈カテーテル等の管理

**対象**: 末梢静脈点滴、中心静脈カテーテル（CVC）管理に伴う観察・IVH（高カロリー輸液等）を含む在宅静脈療法の**実施・観察記録**。

**必須記録項目（観点）**

- 刺入部（挿入部位）の**発赤・腫脹・熱感・疼痛・浸出**の有無
- **滴下速度**／ポンプ設定（可能なら mL/h 等）
- バッグ・シリンジの**残量**（または交換予定）
- **カテーテル固定**状態（松脱、汚染、ドレッシング状態）
- **トラブル**の有無（外れ、逆流、空气関連の訴え、機器アラーム等）と対応
- 必要に応じて：**輸液名・濃度・投与経路**、**無菌操作**の実施、**ラインの開通**確認

**キーワード群（例・グループ別）**

| グループID | 例（いずれかで可） |
|------------|-------------------|
| G_insert_site | 刺入部, 挿入部位, 穿刺部, 固定部, 発赤, 腫脹, 腫れ, 熱感, 疼痛, 浸出液 |
| G_drip | 滴下, 滴下速度, mL/h, ポンプ, 流量, 調整 |
| G_remain | 残量, 残り, バッグ, シリンジ, 交換 |
| G_secure | 固定, ドレッシング, テープ, 松脱, 汚染 |
| G_trouble | トラブル, 外れ, 逆流, アラーム, 閉塞, 血栓, 漏れ |

**充足ルール（チェック用）**: `G_insert_site`・`G_drip`・`G_remain`・`G_secure`・`G_trouble` の**各グループから最低1語**（または明示的な否定文「異常なし」等）を推奨。実務では**「異常なし」一括**より**項目別の否定的記載**が望ましい。

**SOAP 補完の指針**

- **S**: 利用者・家族の訴え（痛み、違和感、機器音）を**そのまま引用**しつつ、刺入部症状の有無を質問で補う形の定型を提案可。
- **O**: 視診触診所見を**部位・程度**で記載（例: 刺入部周囲〜cm、発赤スケールは施設様式に合わせる）。
- **A**: 「ライン管理継続」「感染徴候なし」等は**Oに根拠がある場合のみ**。
- **P**: 次回までの観察重点（固定・残量・滴下）、医師・薬剤師への連絡基準。

**診療報酬・記録上の注意**

- 在宅中心静脈栄養法（IVH）等は**実施内容・観察内容が記録で裏付け**されることが重要。施設・診療所の**様式・必須フィールド**に合わせて項目を増減すること。

### 2.2 `GASTRO_NG` — 胃ろう・経鼻胃管（NG）の管理・注入

**対象**: PEG/胃ろうボタン、経鼻胃管の固定・注入・観察。

**必須記録項目（観点）**

- **固定位置**（**メモリ**／挿入長・外側マーキングの確認）
- 周囲**皮膚状態**（発赤、浸出、肉芽、圧迫、壊死兆候）
- **注入時**の**逆流**・嘔吐・咳嗽の有無
- **内容物**（製剤名、濃度、量、温度、注水の有無）
- **チューブ閉塞**の確認（吸引抵抗、注入抵抗、洗浄結果）
- 胃ろう：**ボタン・外装部**の緩み、漏れ、回転

**キーワード群**

| グループID | 例 |
|------------|-----|
| G_memory | メモリ, 挿入長, マーキング, 目盛, 固定位置, cm |
| G_skin | 皮膚, 発赤, 浸出, 肉芽, 壊死, 刺激, 清潔 |
| G_reflux | 逆流, 嘔吐, 吐き, 誤嚥, 咳嗽, 残餘 |
| G_content | ミルク, 栄養剤, 水分, 量, mL, 温度, 製剤 |
| G_patency | 閉塞, つまり, 洗浄, 吸引, 抵抗, 開通 |

**充足ルール**: 上記5グループそれぞれから**記載または「異常なし」相当**。

**SOAP 補完の指針**

- **O**: メモリ値・注入量・皮膚所見は**測定値で**（推測で埋めない）。
- **P**: 次回注入速度、洗浄頻度、医師報告の閾値。

**診療報酬・記録上の注意**

- 管理材料・在宅栄養管理等の算定がある場合、**実施・観察が個別に追える記録**が求められることがある。事業所コード別ルールを確認。

### 2.3 `FOLEY` — 留置バルーンカテーテル（導尿）管理

**対象**: 尿道留置カテーテル（Foley）の観察・交換補助に関する訪問記録。

**必須記録項目（観点）**

- **尿の性状**（色、混濁、血尿、沈渣所見の言及）
- **尿量**（時間帯別、24h集計、バッグ残量）
- **固定状況**（大腿・下腹部固定、引っ張り、チューブ捻転）
- **外尿道口**の**清潔**状態（腫脹、発赤、浸出）
- バルーン・コネクタの**漏れ**、**疼痛**、**膀胱満脹**の訴え
- 必要時：**膀胱洗浄**の実施有無・出入量

**キーワード群**

| グループID | 例 |
|------------|-----|
| G_urine_quality | 性状, 混濁, 血尿, 色, 透明, 臭気 |
| G_urine_vol | 尿量, mL, cc, バッグ, 時間尿 |
| G_fix | 固定, 引っ張り, 捻転, 大腿 |
| G_meatus | 尿道口, 外尿道口, 清潔, 洗浄, 発赤 |

**充足ルール**: 4グループ＋（トラブル系があれば）**疼痛・漏れ・満脹**のいずれかの言及を推奨。

**SOAP 補完の指針**

- **O**: 尿性状と尿量は**別項目**で書く（監査で読みやすい）。
- **P**: 尿路感染徴候時の連絡、水分摂取・固定再教育。

**診療報酬・記録上の注意**

- 在宅療養管理指導や特定器材に関わる場合、**観察・指導の実体**が記録に残ることが重要。

### 2.4 `PRESSURE_INJURY` — 褥瘡（圧迫創傷）の評価・処置

**対象**: 褥瘡の観察、ドレッシングチェンジ、写真・測定を伴う記録。

**必須記録項目（観点）**

- **部位**（骨突起名、体位で左右）
- **大きさ**（**縦 × 横** cm、可能なら面積）
- **深さ**（**STAGE** または **DESIGN-R** 等、施設で採用する分類）
- **滲出液**の**量**（少量〜大量等の尺度統一）・**性状**（漿液性、血性、膿性等）
- **ポケット**（底面剥離、トンネリング）の**有無**と深さ方向
- **処置内容**（清創の有無、洗浄液、ドレッシング種類、固定）
- 周囲皮膚（浸軟、発赤帯）

**キーワード群**

| グループID | 例 |
|------------|-----|
| G_site | 仙骨, 踵, 坐骨, 大転子, 側臥位, 部位 |
| G_size | cm, 縦, 横, 大きさ, 長径, 短径 |
| G_depth | STAGE, ステージ, DESIGN-R, デザイン, 深さ, 全層 |
| G_exudate | 滲出液, 排膿, 漿液, 血性, 膿性, 滲出量, 少量, 多量 |
| G_pocket | ポケット, 底面剥離, トンネル, 潜行 |
| G_care | ドレッシング, 清創, 洗浄, 交換, フィルム, フォーム |

**充足ルール**: `G_site`・`G_size`・`G_depth`・`G_exudate`・`G_care` は**必須**。`G_pocket` は創の性状上「なし」の明示。

**SOAP 補完の指針**

- **O**: 褥瘡記述は**部位→サイズ→深さ→滲出→ポケット→周囲皮膚→処置**の順が読みやすい。
- **A**: 治癒進行／悪化の判断は**前回比較**があれば明記（数値差）。

**診療報酬・記録上の注意**

- 創傷処置の算定がある場合、**処置の実態・創の状態**が突合しやすい記載が望ましい。

### 2.5 `EOL_CARE` — 終末期ケア（ターミナル期の観察・ケア）

**対象**: ターミナル期・穏やかな死に向けた在宅・訪問の観察・支援記録。

**必須記録項目（観点）**

- **バイタル・全身状態の変化**（意識、呼吸様式、皮膚色調、末梢冷感、血圧脈拍体温は方針に応じ）
- **疼痛**および不快感の**コントロール状況**（スケール、薬剤・用量変更は医師指示の範囲で）
- **家族**への**説明内容**（病状理解、今後の経過イメージ、緊急時対応）
- **精神的ケア**（不安・悲嘆への対応、宗教・文化的配慮の有無）
- **オーラルケア**・体位変換・皮膚ケア等の**コンフォートケア**
- 必要に応じて：**DNAR**・**ACP**の共有状況（事業所方針に従い記載）

**キーワード群**

| グループID | 例 |
|------------|-----|
| G_vital_change | バイタル, 意識, 呼吸, チアノーゼ, 死響, 変化 |
| G_pain | 疼痛, 痛み, NRS, フェイススケール, オピオイド, 鎮痛 |
| G_family | 家族, 説明, 同意, 理解, 不安, 悲嘆 |
| G_psych | 精神的, 傾聴, ケア, 支援, 配慮 |
| G_comfort | 口腔, 体位, 皮膚, コンフォート, 水分 |

**充足ルール**: `G_vital_change`・`G_pain`・`G_family`・`G_psych` の**いずれかが「本日は実施・観察なし」の明示**がある場合を除き、**複数グループの記載**を推奨（終末期は多面的記録）。

**SOAP 補完の指針**

- **S**: 家族の言葉を尊重し、**引用**と**看護師の要約**を分ける。
- **O**: 観察は**時系列**（午前→午後）でもよい。
- **P**: 次訪問までの重点、医師・緩和チームへのエスカレーション。

**診療報酬・記録上の注意**

- 在宅患者緊急訪問看護料等、**緊急性・時間帯**の要件は別途。終末期は**倫理的記録**（尊厳・プライバシー）を最優先。
